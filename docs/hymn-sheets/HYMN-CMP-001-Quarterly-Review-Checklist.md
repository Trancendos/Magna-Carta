## Pre-meeting

- [ ] `python3 scripts/compliance_health_check.py` run — failures assigned

- [ ] ACT-001–012 status exported
- [ ] Participants invited (ISMS, DPO, Eng, SecOps, AI, CAB)
- [ ] MeetingRoom booked

## During meeting

- [ ] DEFSTAN / register score discussed
- [ ] Legislation + obligations registers current
- [ ] STANDARDS-REGISTER + REGULATORS register reviewed
- [ ] SYSTEMS-REGISTER matches AS-BUILT
- [ ] ISO SOA gaps updated
- [ ] SOC 2 evidence schedule on track
- [ ] AI governance (model cards, ACT-004)
- [ ] Supplier DPA register reviewed
- [ ] Pen test / vuln status
- [ ] Risk register top items
- [ ] Tranc3 bridge / ACT-009 noted
- [ ] Actions assigned with owners and dates

## Post-meeting

- [ ] Minutes published (14 days)
- [ ] compliance_action_tracker.yaml updated
- [ ] Register merges within 2 weeks
- [ ] Next review date set (90 days)
