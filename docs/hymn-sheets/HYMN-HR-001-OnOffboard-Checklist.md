# HYMN-HR-001 — On/Offboarding Checklist

**Version:** 1.0.0 · **Procedure:** PROC-HR-001 · **Cookbook:** COOK-HR-001

**Pre-employment**
- [ ] Background check complete (tier appropriate)
- [ ] Agreements signed

**Onboarding**
- [ ] ISMS notified with role and start date
- [ ] Access provisioned (HYMN-IAM-001)
- [ ] Training/attestation started (HYMN-TRN-001)

**Offboarding (same day)**
- [ ] All access revoked
- [ ] Devices/tokens recovered
- [ ] Manager confirmation
- [ ] Ticket closed
