# COOK-CAPA-001 — Corrective Action Playbook

**Version:** 1.0.0  
**Owner:** ISMS Lead  
**Procedure:** [PROC-CAPA-001](../procedures/PROC-CAPA-001-Corrective-Action.md)  
**Hymn sheet:** [HYMN-CAPA-001](../hymn-sheets/HYMN-CAPA-001-CAPA-Checklist.md)

---

## When to use

Audit finding, failed health check (systemic), incident root cause, SLA breach, or management review action.

---

## Open CAPA

1. **Source** — link audit/incident/ACT ID
2. **Root cause** — 5-whys or equivalent; avoid blame-only closure
3. **Owner** and **due date** (P0 ≤ 30 days unless documented exception)
4. Register in ticket system + optional risk register update

---

## Implement

| Type | Example |
|------|---------|
| Corrective | Fix the specific failure |
| Preventive | Update procedure, cookbook, or control |

1. Verify effectiveness after implementation
2. ISMS reviews evidence before close

---

## Close

1. Document evidence of fix and verification
2. Update COMPLIANCE-ACTION-TRACKER if linked ACT
3. Report status at next PROC-CMP-001
