## Purpose

Operational runbooks for **cross-cutting escalation paths** that span multiple procedures. Distinct from [cookbooks](../cookbooks/INDEX.md) (procedure playbooks) and [procedures](../procedures/INDEX.md) (auditable process docs).

| Symbol | Meaning |
|--------|---------|
| ✅ **Programme** | Runbook documented and maintained |
| 🎯 **External validation** | Live execution evidenced (tickets, regulator refs) |

## When to use runbook vs cookbook

| Situation | Use |
|-----------|-----|
| Executing a defined procedure step-by-step | Procedure + cookbook + hymn sheet |
| Sector-specific escalation outside normal IR/DSR | Runbook |
| One-off deployment checklist (cookies, consent banners) | Runbook |
