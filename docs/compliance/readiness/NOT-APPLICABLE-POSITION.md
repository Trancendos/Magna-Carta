# Not-applicable framework position statements

**Owner:** ISMS Lead  
**Last updated:** 2026-06-09  
**Review cycle:** Annual  

Formal position for catalogue frameworks marked **not applicable** in `compliance/frameworks_register.yaml`. These entries remain in the master catalogue for customer and procurement transparency; no certification target or readiness programme is maintained unless scope changes.

| Framework ID | Name | Position |
|--------------|------|----------|
| FW-011 | ISO 50001 (energy management) | No energy-intensive manufacturing or regulated energy baseline; out of organisational scope. |
| FW-028 | Uptime Institute Tiers | Platform hosted on managed cloud (Render/AWS); no owned data-centre tier certification. |
| FW-058 | DoD Impact Level 6 | No US DoD IL6 workloads or classified processing. |
| FW-068 | US ITAR | No export-controlled defence articles or technical data in product scope. |

**Scope change:** If any row becomes in-scope, update `frameworks_register.yaml` applicability, add a dedicated alignment doc, and enable the matching scope signal in `config/magna_carta_config.json`.
